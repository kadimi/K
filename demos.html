Basic Text Input Field

Code

$code = K::input( 'my_txt' );

Output

<input name="my_txt" type="text">

Text Area with Some Attributes and an Initial Value

Code

$code = K::textarea('awesome_textarea'
    , array(
        'class' => 'class_A class_B',
        'placeholder' => 'Type your gibberish here...',
    )
    , array(
        'format' => 'Your message:<br> :textarea'
        'value' => 'How did I get here?',
    )
);

Output

Your message:<br> <textarea class="class_A class_B" name="awesome_textarea" placeholder="Type your gibberish here...">How did I get here?</textarea>
